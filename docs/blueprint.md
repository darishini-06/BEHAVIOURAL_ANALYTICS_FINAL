# **App Name**: FinPulse Insight

## Core Features:

- Project Overview Landing Page: Displays the project title, a concise description of 'Detecting Financial Impulse Behaviour in Young Adults', and a clear call to action.
- Impulse Risk Score Explanation: Presents detailed information regarding how the Impulse Risk Score is calculated, its significance, and underlying methodologies.
- Behavior Profiles Viewer: Allows users to explore predefined financial behavior profiles identified within the young adult demographic, explaining their characteristics and implications.
- Interactive Data Visualizations: Renders 8-10 interactive charts (e.g., bar, line, heatmap, radar, scatter) using Chart.js or Plotly.js to visually represent the research findings based on pre-processed JSON data.
- JSON Data Integration: Handles the loading and parsing of pre-generated JSON data files into the client-side application for powering the interactive charts.
- Personalized Nudge Generation Tool: Provides a tool where users can simulate input or select a behavior profile, and an AI will generate tailored 'nudges' designed to mitigate impulse financial behavior.

## Style Guidelines:

- Light color scheme for clarity and a professional, data-focused presentation, ideal for dashboard viewing. Primary background: #EDF1F4 (HSL: 215, 15%, 90%) for a neutral, clean base that supports data visualization.
- Primary accent color: #437190 (HSL: 215, 50%, 35%)—a sophisticated, deep blue-grey to convey trust and authority in data. Used for key UI elements and graph primary colors.
- Secondary accent color: #32BFB2 (HSL: 175, 55%, 55%)—a vibrant aqua providing a fresh contrast, used for interactive elements, highlights, and differentiating data series on charts.
- All text will use 'Inter', a grotesque sans-serif font known for its high readability and modern, objective aesthetic, suitable for both headlines and detailed chart labels in a dashboard environment.
- Employ a consistent set of clean, minimalist vector icons that support data interpretation and navigation without distracting from the core content, maintaining a 'hackathon-ready' and functional aesthetic.
- Implement a modern, responsive dashboard-style layout with distinct sections for content and interactive charts. Emphasize clear content hierarchy and ample white space to improve data comprehension on various screen sizes.
- Incorporate subtle, performant animations for chart rendering, data updates, and UI transitions. This enhances the interactive experience without adding unnecessary visual clutter, keeping the focus on data insights.
